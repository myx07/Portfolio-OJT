ASSETS FOLDER
=============

images/
  → Place your profile photo here as: photo.jpg
    The HTML references it as: assets/images/photo.jpg
    Recommended size: 400x400px or larger, square crop.
    If the image is missing, a placeholder silhouette is shown automatically.

resume.pdf
  → Place your resume PDF here as: resume.pdf
    The HTML references it as: assets/resume.pdf
    The "Download Resume" button links to this file.
